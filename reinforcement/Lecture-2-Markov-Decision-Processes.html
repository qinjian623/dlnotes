1 MDP定义

MDP定义了一个适合应用RL的环境. 环境完全可见, 与历史无关.

Almost all RL problems can be formalised as MDPs,

具体定义略过.

1.1 组成

  1. 所有的状态集(S)
  2. 状态转移矩阵(P)

2 Markov Reward Process

A Markov reward process is a Markov chain with values.

具体定义:

  1. S 同上
  2. P 同上
  3. R reward function \(R_s = E[R_{t+1} | S_t = s]\)
  4. γ discount factor, \(\gamma \in [0,1]\)

2.1 Return

\(G_t = \sum_{k = 0}^{\infty}\gamma^kR_{t+k+1}\) 其中的discount factor如果是0, 则侧重眼前, 1, 则更加注意整体利益.

2.1.1 为什么使用discount?

  1. 数学上合适1
  2. 避免无限循环
  3. 未来的不确定性没法完全的表示2
  4. 财务上, 立即赚钱相对来说更加划算
  5. 动物和人类也表现的更加喜欢即时的奖励3
  6. 某些情况下适合使用4

2.2 Value Function

the expected return starting from state s:

\(v(s) = E[G_t | S_t = s]\)
